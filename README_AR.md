## 🚀 نظام ERP + PMO احترافي متكامل

نظام إدارة مشاريع متطور (ERP + PMO) مخصص لشركات المقاولات والتطوير العقاري في السعودية.

### ✨ الميزات الرئيسية

#### 📊 لوحات المعلومات
- **Executive Dashboard** - لوحة تنفيذية متقدمة مع رسوم بيانية ثنائية وثلاثية الأبعاد
- **3D Interactive Dashboard** - عرض تفاعلي للمشاريع والبيانات
- **Financial Dashboard** - إدارة مالية متقدمة مع تحليلات الإيرادات والمصروفات
- **Real-time Notifications** - تنبيهات فورية للأحداث الهامة

#### 🏗️ إدارة المشاريع والفيلل
- قائمة شاملة للمشاريع مع متابعة التقدم
- إدارة الفيلل (التخطيط، الإنشاء، الاستكمال، التسليم)
- جداول زمنية متقدمة (Gantt Charts)
- متابعة الميزانية والنفقات لكل فيلا

#### 💼 العقود والفواتير
- إدارة عقود الملاك والمقاولين والموردين
- نظام الفواتير المتقدم
- تتبع الدفعات والمدفوعات
- حفظ واسترجاع المستندات

#### 💰 الإدارة المالية
- تحليل الإيرادات والمصروفات
- حساب الأرباح والخسائر
- تنبؤات مالية
- تتبع التدفق النقدي
- تصنيف التكاليف (مباشرة وغير مباشرة)

#### 📋 مولد التقارير
- تقارير شهرية وفصلية وسنوية
- تقارير مخصصة حسب الاحتياجات
- تصدير بصيغ مختلفة (PDF, Excel)
- إرسال التقارير عبر البريد الإلكتروني

#### 👥 لوحة التحكم الإدارية
- إدارة المستخدمين والأدوار
- نظام الصلاحيات المتقدم
- إحصائيات شاملة للنظام
- تسجيل الأنشطة (Audit Log)

#### 🔔 نظام الإشعارات
- إشعارات فورية للأحداث الهامة
- تصنيفات (نجاح، تحذير، خطأ، معلومات)
- إعدادات الإشعارات الشخصية
- ملخصات يومية

### 🌍 دعم متعدد اللغات
- **العربية (RTL)** - دعم كامل للواجهة العربية
- **الإنجليزية (LTR)** - دعم كامل للواجهة الإنجليزية
- تبديل فوري بين اللغتين

### 🎨 التصميم والواجهة
- **Dark Mode / Light Mode** - موضوعات فاتحة وغامقة
- **Glassmorphism** - تصميم حديث وفاخر
- **Responsive Design** - متوافق مع جميع الأجهزة
- **Smooth Animations** - حركات سلسة وجذابة
- **Modern Components** - مكونات تصميم حديثة

### 🛠️ التقنيات المستخدمة

#### Frontend
- **Next.js 14** - إطار عمل React متقدم
- **TypeScript** - لغة برمجية آمنة
- **Tailwind CSS** - أنماط CSS حديثة
- **Framer Motion** - حركات متقدمة
- **Recharts** - رسوم بيانية تفاعلية
- **Three.js** - رسوميات ثلاثية الأبعاد
- **Zustand** - إدارة الحالة

#### Backend
- **Node.js** - بيئة التطوير
- **Next.js API Routes** - واجهات برمجية
- **PostgreSQL** - قاعدة بيانات قوية
- **Prisma ORM** - إدارة قاعدة البيانات
- **Supabase** - خدمات سحابية

### 🚀 البدء السريع

#### المتطلبات
- Node.js 18+
- npm أو yarn
- PostgreSQL (اختياري)

#### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/nouhshaer5-arch/erp-pmo-system.git
cd erp-pmo-system

# تثبيت المكتبات
npm install

# إنشاء ملف المتغيرات البيئية
cp .env.example .env.local

# تحديث قاعدة البيانات
npm run prisma:migrate

# تشغيل الخادم
npm run dev
```

ثم افتح المتصفح على: **http://localhost:3000**

### 📊 API Endpoints

#### المشاريع
- `GET /api/projects` - جلب جميع المشاريع
- `POST /api/projects` - إنشاء مشروع جديد
- `GET /api/projects/[id]` - جلب مشروع محدد
- `PUT /api/projects/[id]` - تحديث مشروع
- `DELETE /api/projects/[id]` - حذف مشروع

#### الفيلل
- `GET /api/villas` - جلب جميع الفيلل
- `POST /api/villas` - إنشاء فيلا جديدة

#### الفواتير
- `GET /api/invoices` - جلب جميع الفواتير
- `POST /api/invoices` - إنشاء فاتورة جديدة

#### المستخدمين
- `GET /api/users` - جلب جميع المستخدمين
- `POST /api/users` - إنشاء مستخدم جديد

#### المصادقة
- `POST /api/auth/login` - تسجيل الدخول

#### الصحة
- `GET /api/health` - فحص صحة النظام

### 🔐 نظام الصلاحيات

يتم دعم أربعة مستويات من الأدوار:

1. **Admin** (إداري) - وصول كامل لجميع الميزات
2. **Manager** (مدير) - إدارة المشاريع والفيلل
3. **Engineer** (مهندس) - عرض وتحديث البيانات التقنية
4. **Viewer** (عارض) - عرض البيانات فقط
